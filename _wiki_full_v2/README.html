这是什么

PF2 离线百科 是 pf2.huijiwiki.com（Pathfinder 2e 中文维基）的离线镜像，包含 37,000+ 条目：职业 / 祖先 / 背景 / 专长 / 法术 / 怪物 / 物品 / 神祇 / 设定 / 规则。完全本地运行，无需联网。

怎么用

  • 搜索：顶部搜索框，或快捷键 Ctrl+K / /。结果带分类标签（法术/专长/怪物…），可按类型筛选。
  • 浏览：首页左侧「字母浏览」按首字母，或「分类导航」按类型。
  • 列表筛选：法术列表 / 生物总表 / 装备列表等页面顶部有筛选器（按环级/特征/来源…），点按钮即时过滤；大表格自动分页 + 可点表头排序。
  • 目录：长条目右侧有「本页目录」可跳转章节。
  • 收藏：每页右上 ☆ 收藏，侧栏「★ 收藏夹」「⌚ 最近」可回访。
  • 暗黑模式：顶部 🌙 切换，或快捷键 Ctrl+L。
  • 放大图片：点击条目内图片全屏查看。
  • 全部快捷键：按 ? 查看。

自动更新

联网时，有新版本会在顶部弹出提示，可一键自动更新（只下载变化部分，通常几十 KB ~ 几 MB）。也可去 GitHub Releases 手动下载完整包。

提示：手动覆盖更新时，请先完全关闭本程序再解压覆盖，否则正在运行的 pf2-wiki.exe 无法被替换。

已知限制

  • 少数 niche 怪物「特征」页（如卓尔特征）上游维基本身的查询已失效，本地也为空 — 请用「生物总表」或具体怪物页代替。
  • 内容为某次抓取的快照；上游维基的最新编辑需等下次更新包同步。
